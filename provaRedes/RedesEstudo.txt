Camada de Aplicação

Camada de Transporte
	Conexao logica entre processos (fim a fim) 
		faz uma conexao logica entre hospedeiros.
	Protocolos
		TCP, UDP, encapsulamento = agregação de pacotes
	Ampliação
		Detalhamento da entrega hospedeiro a hospedeiro para entregrar processo a processo. Denominada multiplexação/demultiplexação.
			Multiplexação
				Reune os dados proveninete de diferente sockets, diferenciados pelos numero de portas, encapsulando cada parte de dados (payload) com informações do cabeçalho para criar segmentos. Passa esse segmentos então para a camada de rede. E um mecanismo de vários para um.	
			Demultiplexação
				Ocorre no lado da recepçao
				mecanismo de um para vários.
